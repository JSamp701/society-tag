Participants list format:
-----------------------
<string name>,<string email>
<string name>,<string email>

Example:
James Smith,jsmith123@students.bju.edu
Jane Doe,jdoe222@students.bju.edu

Template file information:
-------------------------
This file can be any file you want.  The way the mail system will process it is thus:
%NAME% will be replaced by the player's name
%EMAIL% will be replaced by the player's email
%ID% will be replaced by the player's id
%TARGET% will be replaced by the player's target's name
%SCORE% will be replaced by the player's score
